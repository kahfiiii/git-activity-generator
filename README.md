# 🚀 git-activity-generator

CLI tool untuk mengisi GitHub contribution graph secara otomatis dengan commit yang terlihat natural.

---

## ✨ Fitur Utama

- 📅 Pilih rentang tanggal (custom)
- 📊 Atur jumlah commit per hari
- 🌿 Support branch (main / master / custom)
- ⚙️ UI interaktif di terminal
- 📈 Hasil commit terlihat natural (tidak mencurigakan)
- 📡 Bisa langsung push ke GitHub
